# NLP Analysis of Public Speeches

## Project Overview
This project explores public speech transcripts using Natural Language Processing (NLP)
techniques to understand themes, sentiment, and linguistic patterns.

## Objectives
- Scrape and collect real-world speech transcripts
- Clean and preprocess raw text data
- Perform exploratory text analysis
- Apply sentiment analysis and topic modeling
- Experiment with basic text generation
- Critically evaluate NLP model limitations

## Project Structure
- data/: raw, interim, and processed datasets
- notebooks/: exploratory analysis
- src/: reusable NLP pipelines
- outputs/: figures and tables

## Tools & Technologies
Python, Pandas, NLTK, spaCy, scikit-learn
